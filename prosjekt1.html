LEGO-prosjekt 1

Formål

Lære å bruke for-løkker og Pybricks for å kontrollere LEGO-roboten. Vise tekst på LCD-skjerm, kjøre et rektangel og få roboten til å si «Have a nice day».

Kode


#!/usr/bin/env pybricks-micropython
from pybricks.hubs import EV3Brick
from pybricks.ev3devices import Motor
from pybricks.parameters import Port, Stop
from pybricks.tools import wait


hjernen = EV3Brick()
venstre_motor = Motor(Port.C)
hoyre_motor = Motor(Port.B)


hjernen.screen.print("Hello World")
wait(2000)


def kjør_frem_kort():
venstre_motor.run_angle(2000, 720, Stop.BRAKE, False)
hoyre_motor.run_angle(2000, 720, Stop.BRAKE, True)
